# 简介

## 快速了解服务器

* 这是一个BC群组生存服
* 提供高版本独立区服的 生存玩法 生电玩法 建筑玩法
* 无RPG 无小游戏 无空岛粘液科技   这是个追求**原版**的服务器
* 为防止BUG出现，目前推荐Java版本在**1.18.2**以上
* **全区支持基岩/Java跨端进服**
